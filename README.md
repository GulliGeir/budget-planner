# Iceland Budget Planner

This project aims to provide a simple web tool for planning a trip to Iceland on a budget. Users will be able to input their travel dates, select the region they want to explore, and specify an overall budget. The app will help find or link to relevant resources for flights, accommodation, rental cars, and tours.

## Planned Features

- **Budget Estimator**: form fields where users provide total budget and travel period.
- **Regional Breakdown**: choose from Capital Area, South, East, North, West, and Westfjords to tailor results.
- **Expense Categories**: placeholder sections for flights, accommodation, rental cars, and tours that can later be populated with real data or external links.
- **Interactive Results**: eventually provide cost estimates or recommendations based on user inputs.

This repository currently contains a basic `index.html` implementing the form and result sections to serve as a foundation for further development.
