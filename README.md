# Iceland Budget Planner

This project provides a simple web tool for planning a budget-friendly trip to Iceland. Users can step through a form to enter travel details, view flight options, and then continue to accommodation, rental cars, and tours.

## Features

- **Budget & Date Input**: collect travel dates and overall budget.
- **Interactive Progress Bar**: navigate through Flights, Hotels, Rental Cars and Tours sections.
- **Embedded Widgets**: integrates third party widgets for flights, hotels, cars and activities.

The main page is `index.html` which implements the multi-step planner interface.
